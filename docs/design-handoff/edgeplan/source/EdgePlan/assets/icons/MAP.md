## Loading

Use `lucide-static` from CDN:

```html
<img src="https://unpkg.com/lucide-static@0.469.0/icons/target.svg" width="20" height="20" alt="" />
```

For inline color control (recommended), fetch as SVG and inherit `currentColor`, or use `lucide` web font.
